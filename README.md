📊 React TypeScript Dashboard

A modern admin dashboard UI built with
React, TypeScript, Tailwind CSS, and Recharts.

It includes interactive charts, responsive layout, and reusable components.
<br><br>

✨ Features<br>
⚛️ React + TypeScript setup<br>
🎨 Modern UI with Tailwind CSS<br>
📊 Data visualization using Recharts<br>
Bar Chart<br>
Line Chart<br>
Area Chart<br>
Pie Chart<br>
Radar Chart<br>
Stacked Bar Chart<br>
<br>
📱 Fully responsive design<br>
🧭 Sidebar with toggle functionality<br>
🧩 Reusable components<br>
⚡ Fast development with Vite<br>
<br>
🛠️ Tech Stack
React.js
TypeScript
Tailwind CSS
Vite
Recharts
React Router DOM
